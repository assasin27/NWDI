# Admin Portal Access Guide

## Quick Start

To access the admin portal, use the following credentials:

- **Email:** admin@farmfresh.com
- **Password:** admin123

## How to Access the Admin Portal

1. Navigate to `/login` in your browser
2. Enter the admin credentials listed above
3. Upon successful login, you will be redirected to the admin dashboard

## Features

The admin portal provides access to several key features:

- **Dashboard**: Overview of key metrics and recent activity
- **Products**: Manage product listings (add, edit, delete)
- **Orders**: View and manage customer orders
- **Analytics**: View sales and performance metrics
- **Settings**: Configure portal settings

## Navigation

- Use the sidebar menu to navigate between different sections
- The portal is responsive and works on both desktop and mobile devices
- Click the logout button in the top right corner to sign out

## Security Notes

- These credentials are for testing purposes only
- In a production environment, please use strong, secure credentials
- The admin portal is protected and only accessible to users with admin privileges
